# Explica el flujo completo desde que un admin hace login hasta que puede agregar un plato. Menciona cada archivo que participa y qué hace cada uno.

Primero la petición llega a auth.routes.js que define la ruta, luego pasa al auth.controller.js que recibe la petición, después va al auth.service.js que busca el usuario en la base de datos mediante
auth.model.js, verifica la contraseña con bcrypt.compare() y si todo es correcto genera el token JWT. Una vez con el token, para agregar un plato la petición llega a menu.routes.js que antes de llegar 
al controlador pasa por verifyToken.js que verifica que la firma del token sea válida usando el JWT_SECRET. Si el token es válido continúa al menu.controller.js que llama al plato.service.js y este 
interactúa con la base de datos mediante plato.model.js para guardar el plato.

--------------------------------------------------------------------------------------------------------------------------

# ¿Por qué bcrypt.compare() puede verificar la contraseña si el hash es irreversible? ¿Qué está haciendo internamente?

Esto es porque bcrypt.compare() no necesita descifrar nada. Internamente extrae el salt que está guardado dentro del hash, hashea la contraseña ingresada usando ese mismo salt y compara los dos resultados. 
Si coinciden la contraseña es correcta, todo esto lo hace sin revelar la contraseña original.

--------------------------------------------------------------------------------------------------------------------------

# Si alguien roba un token JWT válido, ¿qué puede hacer con él? ¿Qué mecanismo del sistema limita el daño? ¿Qué agregarías para hacerlo más seguro?

Con el token robado puede hacer peticiones protegidas como agregar platos, modificar precios o incluso eliminar todo el menú mientras el token no expire. Lo que limita el daño es que no puede modificar
el payload para cambiar su email porque necesitaría el JWT_SECRET para recalcular la firma y ese está protegido en el .env. Para hacerlo más seguro agregaría una expiración más corta como 1 hora en lugar 
de 24 horas.
